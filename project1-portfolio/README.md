# Responsive Resume / Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**. The project showcases responsive web design, client-side authentication, theme switching, and local storage functionality without requiring any backend or server.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Sticky navigation with smooth scrolling
- Light and dark mode with user preference saved in Local Storage
- Portfolio login authentication
- Contact form with Local Storage data persistence
- Clean, modern, and user-friendly interface
- No backend or database required

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## Getting Started

1. Clone or download the repository.
2. Open the project folder.
3. Launch `login/login.html` in your web browser.
4. Sign in to access the portfolio.

No installation or server setup is required.
